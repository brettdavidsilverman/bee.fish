Mengenal Pengembangan Web

Ini adalah paragraf contoh untuk menunjukkan tampilan teks pada halaman web. Dokumen ini dibuat menggunakan HTML5 standar dan diformat agar mudah dibaca oleh pengguna internet di Indonesia.

HTML (HyperText Markup Language) berfungsi sebagai kerangka dasar dari sebuah halaman web, sedangkan CSS digunakan untuk mengatur tata letak, warna, dan jenis huruf (tipografi) agar halaman terlihat menarik.

Mengapa Menggunakan Tag Semantik?

  • Mudah Dibaca: Membantu pengembang memahami struktur kode dengan cepat.
  • Aksesibilitas: Mempermudah alat pembaca layar (screen reader) memahami konten bagi difabel.
  • SEO: Mesin pencari seperti Google dapat mengindeks halaman dengan lebih optimal.
